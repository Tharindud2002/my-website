CPU යනු

මධ්‍යම සැකසුම් ඒකකය (Processor / Central Processing Unit) පරිගණකයේ සියලු වැදගත් කාර්යයන් ඉටු කරයි. මෙය Operating System හා Application Software වැනි සියලුම වැඩසටහන් ගබඩා කරයි. CPU එක සතු මෙම විශේෂත්වය නිසා පරිගණකයේ මොළය ලෙස හැදින්වෙයි.

CPU ප්‍රධාන කොටස්

  • ALU (Arithmetic Logic Unit):
    • අංක ගණිත හා තාර්කික කාර්යයන් ඉටු කරයි.
    • එක්තු කිරීම, අඩු කිරීම, ගුණ කිරීම, බෙදීම, සැසදිම ඇතුළත් වේ.
    • තාර්කික ශ්‍රිතවලට දත්ත තෝරා ගැනීම, සංසන්දනය හා ඒකාබද්ධ කිරීම සිදු කරයි.
  • Control Unit:
    • පරිගණක පද්ධතියට උපදෙස් ලබාදීමට විද්‍යුත් සංඥා භාවිතා කරයි.
    • මතකයේ උපදෙස් විකේතනය කර ක්‍රියාත්මක කරයි.
    • ප්‍රධාන කාර්යය: ප්‍රොසෙසරය හරහා තොරතුරු ගලායාම නඩත්තු කිරීම සහ නියාමනය. දත්ත සැකසුම හා ගබඩා කිරීමේ වැඩ වලට සම්බන්ධ නොවේ.
  • Memory / Storage Unit:
    • සාමකාමි ප්‍රවේශ මතකය (RAM) ලෙස හැඳින්වේ.
    • දත්ත, වැඩසටහන් හා අව්‍යාත ප්‍රතිඵල තාවකාලිකව ගබඩා කරයි.

CPU වර්ග හා නිෂ්පාදන

  • Single Core, Dual Core, Quad Core, Hexa Core, Octa Core යනුවෙන් CPU වර්ග ඇත.
  • ඇමරිකාව, තායිවානය, දකුණු කොරියාව, ජපානය, ඉශ්‍රාලය වැනි රටවල් CPU නිෂ්පාදනය කරයි.
  • CPU වේගය මනිනු ලබන්නේ GHz (GigaHertz) මගින්.
