Memory යනු

පරිගණකයේ දත්ත හා තොරතුරු ගබඩා කරන ඒකකය Computer Memory නම් වේ. Memory ධාරිතාව මනිනු ලබන්නේ බිට්(Bit) මගිනි:

  • 8 bit – 1 byte
  • 1024 byte – 1 kilo byte
  • 1024 kilo byte – 1 mega byte
  • 1024 mega byte – 1 tera byte

Computer Memory ප්‍රධාන කොටස්

  • Primary Memory
  • Secondary Memory

Primary Memory

පරිගණකයේ ඇතුළත ක්‍රියාකාරිත්වයට අවශ්‍ය දත්ත ගබඩා කරනු ලබන්නේ Primary Memory කුලය. ප්‍රාථමික මතකය කොටස් 02කි:

  • ROM – Read Only Memory: පරිශිලකයින්ට දත්ත වෙනස් කිරීමට අවසර නොලැබේ. නිෂ්පාදකයා විසින් ROM එකට දත්ත ගබඩා කරයි. ROM සතු ප්‍රධාන කාර්යයන්:
    • පරිගණකය On කරන අවස්ථාවේ සියලුම දෘඪාංග(Hardware) නිවැරදිව ක්‍රියාත්මක වනවාද පරීක්ෂා කිරිම
    • පරිගණකය boot කිරීම (Operating System එක load කිරිම)
  • RAM – Random Access Memory: දත්ත තාවකාලිකව ගබඩා කරයි. පරිගණකයේ සියලුම ලේඛන මතකය ගබඩා කරයි. CPU සමඟ සම්බන්ධව වැඩ කරයි.

Secondary Memory

පරිශිලකයින් නිර්මාණය කරන ලියකියවිලි ගබඩා කරයි. උදාහරණ:

  • Word Document
  • Web Page
  • PowerPoint Slide Presentation

Secondary Memory උපාංග

Hard Disk: Digital දත්ත ගබඩා කිරීම හා ලබා ගැනීම සදහා චුම්බක ද්‍රව්‍ය භාවිතා කරන උපාංගයකි. ධාරිතාව 500GB, 1TB, 2TB වැනි ප්‍රමාණ වලින් ඇත. System Unit එක හරහා Internal drive එකක් ලෙස සම්බන්ධ වේ.

CD/DVD: CD (Compact Disc) හා DVD (Digital Versatile Disc) යනු දත්ත ගබඩා කිරීමේ තැටි වේ. ධාරිතාව CD 750MB, DVD 4-7GB වැනි ප්‍රමාණ වලින් ඇත. System Unit එක හරහා Internal/External Disk Drive එකක් ලෙස සම්බන්ධ වේ.

Pen Drive: දත්ත ගබඩා කිරීමට හා අතේ ගෙන යාමට හැකි කුඩා උපාංගයකි. ධාරිතාව 1GB, 2GB, 8GB වැනි ප්‍රමාණ වලින් ඇත. USB Port හරහා සම්බන්ධ වේ.

External Hard Disk: අමතර දත්ත ගබඩා උපාංගයකි. USB Port හරහා සම්බන්ධ වේ. ධාරිතාව 1TB වැනි ප්‍රමාණ වලින් ඇත.
