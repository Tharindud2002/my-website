මෘදුකාංගය යනු

මෙය පරිගණකයේ ක්‍රියාකාරකම් හා කාර්යභාරයන් පාලනය කරන නියමයන් හා උපකරණවල එකතුවකි. මෘදුකාංගය සම්පුර්ණයෙන්ම අපේෂිත වැඩ කරන ක්‍රම ලේඛනයන්, වැඩ සටහන් හා යෙදුම් වලින් සැදුම් වේ.

මෘදුකාංග උදාහරණ

Microsoft Word (මයික්‍රොසොෆ්ට් වර්ඩ්): Microsoft ආයතනය විසින් නිෂ්පාදනය කරන ලද ලේඛන සකස් කිරිමේ මෘදුකාංගයකි. ලේඛන සකස් කිරීම, සංස්කරණය කිරීම, ආකෘති සහිත ලේඛන හදුන්වාදීම සදහා භාවිතා වේ.

Microsoft Word භාවිතා කිරිමේ කාරයයන්:

  • ලේඛන සකස් කිරිම
  • අකුරු හා පේළි සංරචනය
  • ග්‍රැෆික් හා රූප
  • ප්‍රතිසංස්කරණ හා පුර්වායෝජිත මාදිලි
  • සහයෝගීතාව

Open Source System (ඔපන් සර්ස් පද්ධතිය): සම්පුර්ණ මුලික කේතය පුද්ගලයන්ට හෝ සංවිධාන වලට ලබා දෙන, යෙදුම් මෘදුකාංග හෝ පද්ධති යෙදවුමකි. නිදහස් භාවිතය, පරිශිලක හා සංවර්ධක ආදාන, ආරක්ෂක පද්ධති හා අලුත් වැඩි දියුණු කිරීමේ විශේෂත්ව ඇත. උදාහරණ: Linux, Mozilla Firefox, LibreOffice.

Web Browser (වෙබ් බ්‍රවුසර්): අන්තර්ජාලයේ වෙබ් පිටු, වෙබ් යෙදුම්, අන්තර්ජාල ආශ්‍රිත දත්ත සොයා බැලිම හා ප්‍රයෝජන සදහා භාවිතා කරන මෘදුකාංගයක් වේ. Browser භාවිතා කරන ක්‍රම: වෙබ් පිටු බලන්න, වෙබ් සෙවුම, විද්‍යුත් තැපෑල. උදාහරණ: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, Opera.

Anti-virus (ඇන්ටි වෛරස්): පරිගණක හා ජංගම උපාංග වල අනතුරැදායක මෘදුකාංග හදුනාගනිමින් ඒවා ක්‍රියාත්මක වීමට වළක්වයි. මෙවලම්: වෛරස් හදුනා ගැනීම, මකා දැමීම, ආරක්ෂක යාවත්කාලින කිරීම, ස්කෑන් කිරිම. උදාහරණ: Norton, McAfee, Kaspersky, Bitdefender.
