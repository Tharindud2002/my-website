Excel යනු Microsoft Office හි අංගයක් වන ප්‍රවල ගණකාංග මෘදුකාංගයකි. මෙය වගු (spreadsheets) වශයෙන් දත්ත සංවිධානය කිරීම, ගණනය කිරීම, විශ්ලේෂණය කිරීම හා චාට් සකස් කිරීම සඳහා භාවිතා කෙරේ.

Excel මගින් කළ හැකි දේවල්

  • දත්ත වගු නිර්මාණය (Create tables)
  • සරල හා සංකීර්ණ ගණනය (Simple & complex calculations)
  • සුත්‍ර (Formulas) හා කාර්යයන් (Functions) භාවිතය
  • චාට් (Charts) හා ග්‍රාෆ් (Graphs) නිර්මාණය
  • දත්ත තෝරා ගැනීම හා පෙරහනය
  • වාර්තා සකස් කිරීම

Excel දත්ත පත්‍රය (Data Sheet)

Excel දත්ත පත්‍රය යනු Microsoft Excel මෘදුකාංගයේ පිටුවක් වන අතර, එය පේළි (Rows) සහ තීරු (Columns) වලින් සමන්විත වේ. මෙය සංඛ්‍යාත, පෙළමය සහ දින/වේලා වැනි විවිධ වර්ගවල දත්ත ගබඩා කිරීමට භාවිතා කරයි.

Excel දත්ත පත්‍රයේ අංග

  • Cell (සෙල්): දත්ත ඇතුල් කරන කුඩා කොටසක් (ex: A1, B2). සෙල් එකක් තීරුවක් සහ පේළියක් හමුවීමෙන් බිහි වේ.
  • Row (පේළිය): අංක වලින් නිරූපණය වේ (1, 2, 3…). Excel එකේ දත්ත ඇතුලත් වන තීරුවේ පහළට යන ඉඳුනු පේළි.
  • Column (තීරුව): අකුරු වලින් නිරූපණය වේ (A, B, C…). තීරුවක් කියන්නේ සෙල් එකක් යටින් යටින් යන ස්ථාන.
  • Ribbon (රිබන්): Excel හි ක්‍රමවේද, formatting, chart, formula යනාදිය මෙහි ඇත. File, Home, Insert, Formulas වැනි tab ඇත.

Excel Word Book සෑදීමේ පියවර

  1. Excel File එකක් විවෘත කරන්න (Microsoft Excel හෝ Google Sheets).
  2. Column Headers (තීරු මාතෘකා) සකස් කරන්න.
  3. වචන එක් කිරීම ආරම්භ කරන්න: එක් එක් වචනයකට අදාළ අර්ථය, වාක්‍යයක්, සටහන් ඇතුළත් කරන්න.
  4. අකාරාදී (A-Z / අ-අෆ) ලෙස වචන තීරුව Sort කරන්න.
  5. සෙවුම් පහසුකම් සකස් කරන්න (Optional) - Filter ක්‍රියාත්මක කිරීම.
  6. වර්ණ සහ අකුරු මාරු භාවිතයෙන් වචන වර්ග වෙනස් කරන්න (Optional).
  7. File එක Save කරන්න (උදා: “සිංහල වචන පොත.xlsx”).

Formula සහ Functions භාවිතය

Formula: Excel හි සංඛ්‍යාත්මක ගණනයන් සිදු කිරීම සඳහා ඔබගේ කුසලතාවය අනුව හැඳින්වූ සූත්‍රය.
උදාහරණය: =A1 + B1

Function: Excel විසින් පෙර නිමවා ඇති සූත්‍ර/කාර්යයන්. පහසුවෙන් ගණනය කිරීමට උපකාරී වේ.

Function විස්තරය උදාහරණය
SUM() එකතුව ගණනය කිරීම =SUM(A1:A5)
AVERAGE() මධ්‍යස්ථ අගය =AVERAGE(B1:B5)
MAX() වැඩිම අගය =MAX(C1:C10)
MIN() අඩුම අගය =MIN(D1:D10)
IF() කොන්දේසි පදනම්ව තීරණ ගැනීම =IF(A1>50,"Pass","Fail")

Function Insert කිරීමේ පියවර

  1. සෙල් එක තෝරන්න.
  2. Toolbar එකේ “Insert Function (fx)” ක්ලික් කරන්න.
  3. අවශ්‍ය Function එක තෝරන්න.
  4. Argument (අගයන්) ඇතුළත් කරන්න.
  5. OK ඔබන්න.

Chart යනු?

Excel දත්ත දෘශ්‍ය (Visual) ආකාරයෙන් දැක්වීමට යොදාගන්නා පින්තූරමය නිරූපණයකි. දත්ත හොඳින් වටහාගැනීමට උපකාරී වේ.

Chart වර්ග

  • Column Chart (ස්ථම්භ ප්‍රස්ථාරය)
  • Bar Chart (තාල්පු ප්‍රස්ථාරය)
  • Line Chart (රේඛා ප්‍රස්ථාරය)
  • Pie Chart (පයි ප්‍රස්ථාරය)
  • Area Chart (ප්‍රදේශ ප්‍රස්ථාරය)

Chart සෑදීමේ පියවර

  1. දත්ත තෝරන්න.
  2. Ribbon හි Insert tab එකට යන්න.
  3. “Charts” කොටසින් අවශ්‍ය Chart වර්ග තෝරන්න.
  4. Chart එක worksheet එකට ඇතුළත් වේ.
  5. Chart Design tab එකෙන් Design වෙනස් කළ හැක.
