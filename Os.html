Operating System යනු

අපගේ අවශ්‍යතාවන් ඉටුකර ගැනිමට, එනම් Application Software ධාවනය කිරිමට පරිසරය සකස් කරදෙන unit එක Operating System වේ. පරිශිලකයා හා පරිගණකය දෘඪාංග අතර සම්බන්ධතාවය ඇති කිරිමත් Operating System මඟින් සිදු කෙරේ.

Operating System වර්ග

  • Single User Single Tasking: එක වරකට භාවිතා කල හැකකේ පරිශිලකයා පමණි. එක වරකට කළ හැක්කේ එක වැඩක් පමණි. උදාහරණ: DOS, Android
  • Single User Multi Tasking: එක වරකට භාවිතා කල හැක්කේ එක පරිශිලකයෙකුට පමණි. වරකට කාර්යයන් කිහිපයක් කළ හැකිය. උදාහරණ: Microsoft Windows, Apple Mac OS
  • Multi User Multi Tasking: එක වරකට පරිශිලකයන් කිහිප දෙනෙකුට භාවිතා කල හැකිය. එක වරකට වැඩ කිහිපයක් කළ හැකිය. උදාහරණ: Unity, Linux, Windows Server
  • Real Time Operating System: දෘඪාංග හා මෘදුකාංග සමපත් කළමණාකරනය කිරීමට නිර්මාණය කරන ලද වර්ගයකි. Windows, MacOS හෝ Linux වැනි පොදු OS මෙන් නොව, දැඩි කාල සිමාවන් තුල වැඩ සිදු කිරීමේ හැකියාව ඇත. උදාහරණ: VxWorks, FreeRTOS, RTEMS, ANX, Microlos-Android, Tizen RT

  • ආරක්ෂක කළමණාකරණය

Operating System අතුරුමුහුණත්

  • CLI – Command Line Interface: අතිතයේ පරිගණකවල ක්‍රියාත්මක වූ අතුරුමුහුණත. විධානය මතක තබා ගැනීම සිදු වේ. උදාහරණ: MS-DOS
  • GUI – Graphical User Interface: වර්තමාන මෙහෙයුම් පද්ධතියේ අතුරුමුහුණත. විධානයක් රූපමය ආකාරයෙන් සිදු වේ. උදාහරණ: Windows
